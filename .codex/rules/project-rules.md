# Project Rules

- Keep PasteVLv personal and standalone; do not add company, ERP, or workplace
  references.
- Preserve the Mac-only scope unless the user explicitly changes it.
- Prefer native macOS APIs over cross-platform abstractions.
- Keep generated artifacts out of version control.
- Update `README.md`, `docs/ARCHITECTURE.md`, and these `.codex` docs when a
  change alters run commands, architecture, permissions, or user-facing behavior.
- Treat clipboard data as sensitive. Do not add logging of copied content.
