# Codex Init

This directory contains repo-native guidance for Codex and other AI agents.

## Files

- `project-brief.md`: product intent, non-goals, and current implementation.
- `rules/`: durable project rules that should be read before edits.
- `skills/`: reusable playbooks for repeated work.
- `agents/`: role prompts for focused implementation or review passes.
- `../docs/commands.md`: shared command reference for humans and agents.

## Recommended Load Order

1. `AGENTS.md`
2. `.codex/project-brief.md`
3. `.codex/rules/project-rules.md`
4. `.codex/rules/macos-native.md`
5. `.codex/rules/git-and-validation.md`
6. `.codex/skills/paste-vlv-development/SKILL.md` when changing the app

Keep these docs concise and update them when workflow, architecture, or commands
change.
