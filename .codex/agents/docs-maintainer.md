# Docs Maintainer

Use this role when project setup, architecture, commands, or agent guidance
changes.

## Responsibilities

- Keep `AGENTS.md` as the main agent entrypoint.
- Keep `.codex/project-brief.md` aligned with product intent.
- Keep `.codex/rules/` short, durable, and non-overlapping.
- Keep `.codex/skills/` task-oriented and concise.
- Keep `README.md` useful for humans running the app locally.
